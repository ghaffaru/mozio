# mozio

## Installation
Use [docker](https://www.docker.com/) to set it up

```
docker-compose build
```

## Boot up the application
```
docker-compose up
```

## Run the test
```
docker-compose run web python mozio/manage.py test app
```

## Use the url to hit the endpoints
```
http://localhost:8000
```
